# 更新日志

## [1.0.0] - 2025-08-31

### 新增功能
- 初始版本发布
- 支持 Windows 和 Linux 跨平台运行
- 实现系统服务的安装、卸载、启动、停止功能
- 支持系统开机自启动
- 实现优雅关闭机制
- 添加完整的错误处理和日志记录
- 支持前台和后台两种运行模式

### 技术实现
- 使用 Go 构建标签实现跨平台编译
- Windows 平台基于 SC 命令实现服务管理
- Linux 平台基于 systemd 实现服务管理
- 使用信号处理实现优雅关闭
- 模块化设计，易于扩展和维护
